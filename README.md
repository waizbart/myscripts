# myscripts

Collection of scripts and tools for server setup and automation.

## Scripts

| Script | Description |
|---|---|
| [bootstrap-nginx-ssl](bootstrap-nginx-ssl/) | Go CLI tool that bootstraps an Ubuntu server with Docker containers behind Nginx reverse proxy + Certbot SSL |
| [ralph-loops](ralph-loops/) | Autonomous AI agent loop that executes user stories from a PRD one at a time, with tests and commits per iteration. Works with Claude Code and Amp. |
